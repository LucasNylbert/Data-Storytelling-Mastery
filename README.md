# 📊 Data Storytelling Mastery  
### Transformando Dados em Narrativas que Geram Decisão

---

## 🎯 1. Contexto e Objetivos

Este repositório foi desenvolvido como parte de um estudo prático sobre **Storytelling com Dados**, com foco em transformar análises técnicas em **narrativas visuais claras, objetivas e orientadas à tomada de decisão**.

A base teórica do projeto combina os princípios de:
- Storytelling com Dados (Cole Nussbaumer Knaflic)
- Visualização de Dados (Edward Tufte)
- Design e Comunicação Visual de Dados

---

### 🎯 Objetivos do projeto

- Aplicar os 6 pilares do Storytelling com Dados:
  - Contexto claro
  - Escolha correta de gráficos
  - Redução de saturação visual
  - Direcionamento de atenção
  - Pensamento de design
  - Construção de narrativa

- Melhorar a comunicação de insights analíticos
- Reduzir ruído visual em dashboards
- Criar um modelo reutilizável de estrutura de storytelling

---

## 📚 2. Curadoria de Fontes

As seguintes fontes foram utilizadas como base teórica:

- 📘 Storytelling com Dados — Cole Nussbaumer Knaflic

## 🧠 3. Engenharia de Prompts e Aprendizados (Cicatrizes)

Nesta seção, documentamos o processo de interação com IA para refinamento das análises.

---

### 🔍 Prompt 1 — Transformação de gráfico saturado
> “Como transformar um gráfico de barras saturado em uma visualização baseada em storytelling?”

✔ Aprendizado:
- Uso de cores neutras (cinza) para dados secundários
- Destaque apenas no insight principal
- Redução de elementos visuais desnecessários

---

### 🔍 Prompt 2 — Atributos pré-atentivos
> “Como usar atributos pré-atentivos para guiar a atenção do usuário?”

✔ Aprendizado:
- Cor, tamanho e posição são mais importantes que legenda
- O olhar humano segue contraste e hierarquia visual

---

### ⚠️ Dificuldades encontradas

- Tendência da IA em sugerir paletas “coloridas demais”
- Uso incorreto de gráficos 3D em respostas iniciais
- Falta de hierarquia visual em dashboards complexos

---

### 💡 Aprendizado crítico

> Gráficos 3D distorcem percepção e devem ser evitados em análise de dados.

---

## 📊 4. Miniguia de Estudo (Entrega Final)

---

### 🧠 Resumos estruturados

- Storytelling com dados não é estética — é clareza de decisão
- Um gráfico deve transmitir apenas uma ideia principal
- O uso de cor deve guiar o olhar, não decorar o gráfico
- Menos elementos = mais entendimento

---

### 📌 Princípios fundamentais

- 🔹 Use o cinza como base neutra
- 🔹 Destaque apenas o insight principal
- 🔹 Evite excesso de legendas e grids
- 🔹 Priorize comparação e contexto
- 🔹 Simplifique sempre

---

## 📖 Glossário

- **Atributos pré-atentivos:** elementos visuais percebidos instantaneamente (cor, tamanho, posição)
- **Carga cognitiva:** esforço mental necessário para interpretar um gráfico
- **Taxa de tinta de dados:** proporção entre informação útil e elementos decorativos
- **Insight:** interpretação relevante de um dado
- **Storytelling:** construção de narrativa com base em dados
